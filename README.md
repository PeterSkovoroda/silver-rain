Dependencies
------------
* Python 3
* GStreamer 1.0
* GStreamer 1.0 plugins good
* GTK+3

Installation
------------

### Compiling from source using autotools

Requires gnome-autogen.sh

Get the code:

    git clone https://github.com/PetrSkovoroda/silver-rain.git && cd silver-rain

Compile and install:

    ./autogen.sh
    make
    sudo make install
